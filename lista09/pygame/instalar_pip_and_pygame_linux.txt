#Primeiro tenha o python3 já instalado.

#----Instale o módulo pip do python----

Abra o terminal e digite o seguinte comando:
	sudo apt-get install python3-pip

Pode ser que o pip venha desuatalizado, portanto digite:
	pip install -U pip
	pip install --upgrade pip
	
#----Instalar Pygame----	
Com o pip instalado digite o seguinte comando:
	pip install pygame --user
